# Julie Anne Cantillep Portfolio

Scrapbook-inspired portfolio built with Next.js and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Pages

- `/` Home
- `/about` About Me
- `/projects` Projects
- `/skills` Skills
- `/journey` Journey
- `/journal` Journal / Blog
- `/contact` Contact
- Custom 404 page

## Replace the portrait

Put your own image beside this script before running it and name it one of these:

- `profile.jpg`
- `profile.jpeg`
- `profile.png`

The script stores it in `src/assets/images` and also copies it to `public/avatar.jpg` so the site can display it.
